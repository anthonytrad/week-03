After downloading the theme from Themeforest, you need to unzip it, and you should see the following structure:

  • Demo – contains the demo data XML file.
  • Documentation – the theme documentation in PDF format.
  • thewall.zip – the zipped theme (this will be the file you will upload to install the theme).
  • Assets – the theme’s PSD files.

Installing the theme

We will assume your site has WordPress already installed. If you need help installing WordPress, you can find detailed instructions in WordPress’ codex page. Once your site is ready, we can begin installing The Wall.

  • On your WordPress administrator, navigate to Appareance > Themes.
  • Click on Add New and then click Upload.
  • Choose the thewall.zip file (the one that’s inside the folder where you uncompressed the The Wall file you downloaded), and upload it.
  • Now click on Install Now. If all goes well, this process shouldn’t take more than a few minutes, depending on the speed of your internet connection. Once the theme is installed, you should see this message: Theme installed successfully.
  • Now click the Activate button.

And that’s it! The Wall is now installed, activated and ready to use!

Installing the required plugins

The Wall requires you to install some plugins in order to work. But luckily this is very easy to do!

After you activated The Wall, you should see this message on the top of your WordPress admin page:

  • This theme requires the following plugins: Explora Widgets, Explora Types and Explora Metaboxes.
  • This theme recommends the following plugins: Contact Form 7, I Recommend This ans WPBackery Visual Composer.

Themes are not supposed to provide extra functionality, so all the awesome features The Wall offers need to be separated from the theme. This is why we provide this functionality in the form of plugins, which need to be manually installed by you. To do this, simply follow the instructions below.

  • Click the Begin installing plugins link.
  • On the screen that appears, you will see Install buttons for all the bundled plugins. Install the required plugins and the recommended ones that you want.
  • Just like the theme itself, the plugins need to be activated also. The Install buttons should have disappeared, and there should be Activate buttons instead. Just activate all the plugins you installed and you are good to go!

If you have already installed the theme and the required plugins, you can now begin building your site! The Wall is a really simple theme so luckily there's not much to learn about it if you are familiar with WordPress.

Creating the homepage

The homepage basically consists of a slider and a wall. To create it, follow the steps below.

  • Navigate to Pages > Add New.
  • Choose the Homepage template from the Template select on the Page Attributes box.
  • Enter a title for the page and click Publish.
  • Now you need to set page as the Front Page. To do this, navigate to Appearance > Reading and set the Front page displays option to A static page. Then select your homepage on the Front page field. And that’s it! Now you just need to create your homepage’s gallery and you are all set.

Now that you have created your homepage, you can create its slider and wall.

Creating the Homepage's Slider

After clicking the Publish button, a Homepage Slider box will appear. The first two options that you will see are Slideshow and Interval. This allows you to define wether the slider will animate automatically or not, and the interval of time each slide will be visible.

Below, you will see an empty list of slides. Click the camera icon to add an image to a slide (on the Add Media popup that will appear, use the Insert into post button to set the image). You can also set a Title, Description, Link and Date to the slide. Also, you can set the Image Display option, which allows you to decide wether the Cover or Contain methods will be used to display the photo on the slider. The Cover method makes sure the photo covers the entire viewscreen, even if a portion of the image needs to be cropped, while the Contain option makes sure the picture is not cropped at all.

Creating the Homepage's Wall

Creating your wall is pretty straightforward. Just use the Add Gallery, Add Album, Add Blog Post and Add Text Block buttons to add modules to the wall. Then you can rearrange them by dragging and dropping the modules with your mouse.

Blog Page

This is the page where your blog posts will be featured. To create it, simply follow the instructions below.

  • Navigate to Pages > Add New.
  • Enter a title for the page (for example: “Blog”) and click Publish.
  • Now you need to set page as the Posts Page. To do this, navigate to Appearance > Reading and set the Front page displays option to A static page. Then select your blog page on the Posts page field.

You can now begin adding Blog Posts to your site.

Custom Pages

Custom pages are multi purpose pages that can be used for displaying any kind of content on your site. Follow the instructions below to create one.

  • Navigate to Pages > Add New.
  • Enter a title for the page and click Publish.

Now you can begin adding content to your page using the Visual Composer editor. If you need help using the Visual Composer plugin, head to the plugin’s knowledge base.

Contact Page

Contact pages are used to display a page where your site visitors can see your contact information, location map and a contact form so your visitors can contact you. To create a contact page, follow the instructions below.

  • Navigate to Pages > Add New.
  • Choose the Contact template from the Template select on the Page Attributes box.
  • Enter a title for the page (for example: “Contact”) and click Publish.

Now you need to configure your contact form using the Contact Form plugin. Navigate to the Contact menu item on your WordPress’ administrator. There should be a form already created so you can use that one or create a new one. In any case, here is the code you need to enter in the Form section of your contact form:

<section class="contact">
  <h2>Leave a Comment</h2>
  <p>Your email address will not be published. Required fields are marked <em>*</em></p>
  <div>
    <p class="form-name field">[text* your-name placeholder "NAME"]</p>
    <p class="form-email field">[email* your-email placeholder "EMAIL"]</p>
    <p class="form-web field">[email* your-email placeholder "WEBSITE"]</p>
    <p class="form-comment">[textarea your-message][submit "SEND"]</p>
  </div>
</section>

Now you need to insert the contact form in your contact page. To do this, copy the contact's form shortcode and paste it into your contact page's content. It should look something like this:

[contact-form-7 id="###" title="Contact form 1"]

Albums

Albums are collections of photos which have their own page and can be featured on the Homepage's Wall. Follow the instructions below to create one.

  • Navigate to Albums > Add New.
  • Enter a title for the album and click Publish.

Now you can begin adding photos and descriptions to your album. Here's the list of fields you have available:

Headline
This is the handwritting styled headline that appears on top of your album's title.
Cover Photo
This is the cover photo that appears on top of the page.
Heading Text
This is the text that appears before the gallery
Gallery
This is the album's photo gallery.
Description
This is the description that is displayed on the album's sidebar.
Wall description
This is an alternative optional description that you can use if you want to have a different description for this album when diplayed on your homepage's wall.
Album Page layout
Choose the positioning of your sidebar for this album.
Gallery Columns
Choose the number of columns for this album.
Meta Info
This is the meta info that is displayed on the album's sidebar.

Importing the demo data

We have included an XML file that allows you to recreate Thw Wall's demo site. If you choose to do this, posts and pages will be created on your site for you. Then you can add or remove the content on your site to fit your needs.

  • To import the demo data, go to Tools > Import and choose WordPress on the next screen. You may have to install the WordPress importer if you don’t have it already.
  • Choose the XML data file (thewall_demo.xml) that’s included in the folder named Demo
  • Click on Upload file and import
  • Check the Download and import file attachments box
  • Click on Submit

Due to WordPress Importer limitations, the Homepage's content cannot be imported, and you will need to create it manually.

Please note that as stated on Munich’s product page, the images are not included. We have bought licenses to use the images on our demo site, but we do not have the rights to give them away.

Customizing your site

If you want to customize your site to make it more unique, we've got good news for you! We have included over a hundred customization options so most elements of The Wall are customizable.

To make the customization process really easy and intuitive, The Wall uses the WordPress Customization API, so you can see how the changes you make affect your site live! Without the need to reload your site every time you make a change.

To customize your site, use the Customize tool which you can find in Appearance > Customize

Below there is a brief description of each of the available options sections.

Customization Options Sections

General

  • Logos: Choose your site's logo.
  • Sidebars Visibility: Choose your sidebar's visibility options.
  • Colors: Choose your site's colors.

Page Backgrounds

  • Backgrounds: Choose the backgrounds for each of the pages you have created.

Social & Sharing

  • Social networks links: Enter the URL's for your social network links.
  • Posts sharing buttons: Choose the social sharing buttons.

Shop

  • List of products: Choose the number of columns and image quality for your shop archive page.

Custom Code

  • Analytics: Enter your Analytics Code here.
  • Custom CSS: Enter any custom CSS code here.
  • Custom Scripts: Enter any custom scripts here.

Site Identity

  • Site Title: Your site's title.
  • Tagline: Your site's tagline.
  • Site Icon: Your site's favicon.
